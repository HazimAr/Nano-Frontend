## Setup ---------------------------------------------------------

## install dependencies `npm i` or `yarn`
## install prettier globally `npm i -g prettier`
## ask devs for files below:
.env.development
.env.local
.env.production
next.config.js
## Commands
`npm run dev`
w/Prettier `npm run clean`
w/Jest `npm run test`
Build `npm run build`
